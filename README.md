# Portfolio Website Starter

This folder is ready to upload to your GitHub repository.

## Folder structure

```text
portfolio-website-starter/
│
├── index.html
├── projects.html
├── about.html
├── contact.html
├── style.css
│
├── images/
│   ├── profile/
│   │   └── profile-photo.jpg
│   └── projects/
│       ├── hero-image.jpg
│       ├── project-1-cover.jpg
│       ├── project-2-cover.jpg
│       └── project-3-cover.jpg
│
└── files/
    └── resume.pdf
```

## Where to add things

- Put your homepage large image in:
  `images/projects/hero-image.jpg`

- Put your project cover images in:
  `images/projects/`

- Put your profile photo in:
  `images/profile/profile-photo.jpg`

- Put your resume PDF in:
  `files/resume.pdf`

## Important

Keep the file names exactly the same unless you also change them inside the HTML files.

## How to upload to GitHub

Upload everything inside this folder to the root of your repository, not inside another extra folder.

Your repo should finally look like this:

```text
yourusername.github.io/
├── index.html
├── projects.html
├── about.html
├── contact.html
├── style.css
├── images/
└── files/
```
